# 2026-07-22

## 分析项目并新增 integration_test 流程测试
- 项目本质：Flutter/Dart 酷狗音乐客户端，GetX + just_audio + get_storage + dio + freezed + sqflite。PC 端走 `HomePage`(侧边栏 + LocalRouter 内部 Navigator)，移动端走 `MobileLayoutPage`。
- 关键交互点梳理：侧边栏 `Sidebar` 菜单(首页/广场/排行榜/最近播放/我的)；搜索由 `TitleSearchField.onSubmitted` 经 `LocalRouterController.navigateTo('/search', {initialKeyword})` 进入 `SearchPage`；"我的"在 `userInstance.token` 为空时跳 `/login`；底部控制条 `BottomControl` 控制播放/切歌/循环；播放页 `MusicPlayerScreen` 由 `Get.toNamed('/player')` 进入。
- 新增 `integration_test/app_flow_test.dart`（5 组：启动&侧边栏、侧边栏导航、登录流程、搜索流程[Mock]、播放控制）与 `integration_test/mock/search_mock_adapter.dart`（dio 适配器，仅 Mock `/search/complex`）。
- `pubspec.yaml` 增加 `integration_test: sdk: flutter`；`flutter pub get` 通过；`flutter analyze integration_test` 无 error/warning。
- 运行需 Android/iOS 模拟器或真机：`flutter test integration_test/app_flow_test.dart`（集成测试跑在真实设备，跳过 `main()` 的 `JustAudioBackground.init`，强制 1280x800 桌面布局）。
